# Security Policy

## Supported Versions

We currently provide security updates for the latest release on the default branch.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

Email: deonmenezes@example.com

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Affected versions
- Suggested remediation (if any)

We aim to acknowledge reports within 48 hours and will coordinate disclosure once a fix is available.
